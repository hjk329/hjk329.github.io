## Github 패키지

디자인 시스템을 구축하는 프로젝트를 시작하게 됐다.   
private 프로젝트를 사용하면서 무료로 사용할 수 있는 조건에 따라 Github 패키지를 사용해보기로 했다!   

공식 문서의 QuickStart 따라 스텝별로 진행했다.  
4번까지 따라한 후 `npm install`을 하면 `package.json` 이 생성된다. (오오오!)
